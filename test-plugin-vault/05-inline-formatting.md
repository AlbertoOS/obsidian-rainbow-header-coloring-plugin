# 05 — Inline Formatting in Headings

All of these should be colored. Verify that inline formatting inside a heading
does not break the background or text color.

## **Bold heading** — should be colored

### _Italic heading_ — should be colored

#### **Bold** and _italic_ mixed — should be colored

##### `Inline code` inside a heading — should be colored

###### ~~Strikethrough~~ inside a heading — should be colored

# [External link in heading](https://obsidian.md) — should be colored

## [[Wikilink in heading]] — should be colored

### ![Embedded image alt](attachment.png) — should be colored (even if image missing)

#### Heading with ==highlighted== text — should be colored

##### Heading with $inline math$ — should be colored

###### Emoji in heading 🌈 — should be colored
